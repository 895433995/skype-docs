Prerequisites

  1. Bootstrap Skype Web SDK
  2. Sign-In

Provide SIP URIs to create a group chat conversation with the Conversation Control.

The Conversation Control provides a method, renderConversation(...) that takes a container element and configuration options and creates a visual control similar to a client chat window.

The configuration options include specifying modalities and a list of participants to invite.

Send a message to invite participant(s).

The Conversation Control handles sending invitations to invite participant(s) to the conversation. It also provides the ability to add additional participant(s).

When finished click the end button to terminate the conversation.
