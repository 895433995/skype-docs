Select a contact to remove them from the contact list.

A group object exposes a persons collection which allows us to remove a contact from that group. In this example we subscribe to the persons collection on the all property of the personsAndGroupsManager object as well as listening for the added event. In some cases a displayName property may not be readily available so we can listen to the changed event of the displayName property to be certain we have the value.
