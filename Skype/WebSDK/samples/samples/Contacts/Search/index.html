Prerequisites

  1. Bootstrap Skype Web SDK
  2. Sign-In

Provide a SIP URI or contact name to search.

The personsAndGroupsManager exposes createPersonSearchQuery() that creates a search query object. This object requires a text property be set with the intended search query and optionally a limit can be chosen to shrink the results set. In this example a limit of 5 has been set for demonstration purposes. Once these values have been set we can call getMore() on the search query object and check the results collection of the search query object to see if any contacts were found.
